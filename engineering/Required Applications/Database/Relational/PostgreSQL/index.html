پایگاه‌داده PostgreSQL

می‌توانید با استفاده از فایل docker-compose زیر به راحتی با استفاده از داکر پایگاه‌داده‌ی PostgreSQL را راه‌اندازی کنید:

docker-compose.yml
services:
  postgres:
    image: postgres:16.4
    ports:
      - 5432:5432
    environment:
      - POSTGRES_PASSWORD=!@#123qwe
      - POSTGRES_USER=postgres
    volumes:
      - ./data:/var/lib/postgresql/data

هم‌چنین برای اتصال به این پایگاه داده می‌توانید از PGAdmin استفاده کنید که هم به صورت local و هم به صورت داکری قابل نصب است.
